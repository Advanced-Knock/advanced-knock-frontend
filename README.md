## Overview

Advanced Knock Frontend is a Next.js web application providing product pages, marketing content, and checkout integration for Advanced Knock.

---

## Tech Stack

- **Next.js:** ^15.1.0
- **React:** ^19.0.0
- **TypeScript:** 5.3.0
- **TailwindCSS:** Styling
- **Framer Motion:** Animations
- **Stripe:** Checkout integration

- ✅ Education content pages

## Quick Start

### Prerequisites

- Node.js 20+
- npm or pnpm

### Installation

```bash
# Clone repository
git clone https://github.com/Advanced-Knock/advanced-knock-frontend.git
cd advanced-knock-frontend

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your configuration

# Start development server
npm run dev
```

---

## Development

### Running the App

```bash
# Development mode
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Pages

- `/products/advanced-knock` - Product landing page
- `/products/advanced-knock-education` - Education page
